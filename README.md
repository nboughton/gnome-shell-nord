# How to use

```
mkdir -p ~/.themes/default-nord/gnome-shell
cd ~/.themes/default/nord/gnome-shell
git clone https://github.com/nboughton/gnome-shell-nord .
```

Then open GNOME tweaks and set your shell theme to default-nord
